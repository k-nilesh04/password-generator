# 🔐 Password Generator

A modern, responsive Password Generator built with **React**, **Vite**, and **Tailwind CSS**. Generate secure and customizable passwords instantly with an intuitive user interface.

---

## 🚀 Live Demo

> **Live Website:** https://password-generator-sandy-zeta.vercel.app

---

## ✨ Features

* 🔢 Select password length
* 🔠 Include Uppercase Letters
* 🔡 Include Lowercase Letters
* 🔣 Include Numbers
* 💥 Include Special Characters
* 📋 One-click Copy to Clipboard
* ⚡ Instant password generation
* 📱 Fully Responsive Design
* 🎨 Modern UI with Tailwind CSS

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/k-nilesh04/password-generator.git
```

Navigate to the project folder

```bash
cd password-generator
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```

---

## 📖 How It Works

1. Choose the desired password length.
2. Select the character types you want:

   * Uppercase
   * Lowercase
   * Numbers
   * Symbols
3. Click **Generate Password**.
4. Copy the generated password using the copy button.
